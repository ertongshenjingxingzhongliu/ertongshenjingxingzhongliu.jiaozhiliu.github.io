胶质瘤

1. 概念

儿童胶质瘤是一种起源于儿童中枢神经系统（脑和脊髓）中神经胶质细胞的肿瘤。神经胶质细胞是大脑的支持细胞，负责维持神经元的正常功能。胶质瘤是儿童最常见的原发性脑肿瘤，占所有儿童中枢神经系统肿瘤的40%~50%。与成人胶质瘤相比，儿童胶质瘤在生物学行为、病理类型和预后方面存在显著差异。

儿童胶质瘤可发生在脑的任何部位，常见于小脑、脑干、视神经通路和大脑半球。其发病机制复杂，可能与遗传因素、环境因素及基因突变有关。

2. 疾病分类

根据世界卫生组织（WHO）的中枢神经系统肿瘤分类，儿童胶质瘤可分为以下级别和类型：

（1）按恶性程度分级

  • Ⅰ级（低级别胶质瘤）：生长缓慢，边界清晰，预后较好。常见类型包括毛细胞型星形细胞瘤（Pilocytic Astrocytoma）和室管膜下巨细胞星形细胞瘤（Subependymal Giant Cell Astrocytoma）。
  • Ⅱ级（低级别胶质瘤）：生长较慢，但可能进展为高级别胶质瘤。常见类型包括弥漫性星形细胞瘤（Diffuse Astrocytoma）和少突胶质细胞瘤（Oligodendroglioma）。
  • Ⅲ级（高级别胶质瘤）：恶性程度较高，生长较快。常见类型包括间变性星形细胞瘤（Anaplastic Astrocytoma）和间变性少突胶质细胞瘤（Anaplastic Oligodendroglioma）。
  • Ⅳ级（高级别胶质瘤）：恶性程度最高，侵袭性强。常见类型包括胶质母细胞瘤（Glioblastoma Multiforme, GBM）和弥漫性中线胶质瘤（Diffuse Midline Glioma, DMG）。

（2）按部位分类

  • 小脑胶质瘤：常见于儿童，多为低级别胶质瘤（如毛细胞型星形细胞瘤）。
  • 脑干胶质瘤：多为高级别胶质瘤（如弥漫性内生性脑桥胶质瘤，DIPG），预后较差。
  • 视神经通路胶质瘤：常见于神经纤维瘤病（NF1）患儿，多为低级别胶质瘤。
  • 大脑半球胶质瘤：可表现为低级别或高级别胶质瘤。

3. 病因

儿童胶质瘤的具体病因尚不明确，但可能与以下因素有关：

（1）遗传因素

  • 基因突变：某些基因突变（如BRAF、IDH1、TP53等）与胶质瘤的发生密切相关。
  • 遗传综合征：某些遗传性疾病患儿胶质瘤发病率较高，例如神经纤维瘤病1型（NF1）、结节性硬化症（TSC）和Li-Fraumeni综合征。

（2）环境因素

  • 电离辐射：暴露于高剂量电离辐射（如放疗）可能增加胶质瘤风险。
  • 化学物质：某些致癌物质（如亚硝酸盐）可能与胶质瘤发生有关。

（3）其他因素

  • 病毒感染：某些病毒（如巨细胞病毒）可能与胶质瘤发生有关。
  • 免疫因素：免疫系统功能异常可能参与肿瘤发生。

4. 临床症状

儿童胶质瘤的症状与肿瘤部位、大小、生长速度及患儿年龄有关，常见症状包括：

（1）颅内压增高

  • 头痛：早晨加重，可能伴有恶心、呕吐。
  • 视乳头水肿：视力下降或视野缺损。
  • 意识改变：嗜睡、烦躁或昏迷。

（2）神经功能障碍

  • 运动障碍：肢体无力、行走不稳。
  • 感觉异常：麻木、刺痛。
  • 语言障碍：表达或理解困难。

（3）癫痫发作

  • 部分性或全身性癫痫发作。

（4）部位特异性症状

  • 小脑胶质瘤：共济失调、平衡障碍。
  • 脑干胶质瘤：吞咽困难、面瘫、复视。
  • 视神经通路胶质瘤：视力下降、眼球突出。
  • 大脑半球胶质瘤：偏瘫、性格改变。

5. 诊断依据

儿童胶质瘤的诊断需结合临床表现、影像学检查和病理学检查：

（1）影像学检查

  • 颅脑CT：显示肿瘤位置、大小及钙化、出血等特征。
  • 磁共振成像（MRI）：首选检查方法，可清晰显示肿瘤范围及与周围组织的关系。
  • 多模态MRI：包括弥散加权成像（DWI）、灌注加权成像（PWI）等，用于评估肿瘤代谢和功能。
  • 正电子发射断层扫描（PET）：评估肿瘤代谢活性。

（2）病理学检查

  • 活检或手术切除：获取肿瘤组织进行病理学诊断，明确肿瘤类型和分级。
  • 分子病理学检测：检测BRAF、IDH1等基因突变，指导治疗和预后评估。

6. 治疗方法

儿童胶质瘤的治疗需根据肿瘤类型、部位和患儿年龄制定个体化方案，主要包括：

（1）手术治疗

  • 完全切除：适用于低级别胶质瘤，可显著改善预后。
  • 部分切除：适用于高级别胶质瘤或位于重要功能区的肿瘤，以缓解症状和降低肿瘤负荷。

（2）放射治疗

  • 适应症：高级别胶质瘤或术后残留的低级别胶质瘤。
  • 技术：三维适形放疗（3D-CRT）或调强放疗（IMRT）。

（3）化学治疗

  • 适应症：低级别胶质瘤（尤其婴幼儿）或复发胶质瘤。
  • 常用药物：卡铂、长春新碱、替莫唑胺（TMZ）等。

（4）靶向治疗

  • 适应症：复发或难治性胶质瘤。
  • 常用药物：BRAF抑制剂（如达拉非尼）、MEK抑制剂（如曲美替尼）。

（5）免疫治疗

  • 临床试验：包括免疫检查点抑制剂、CAR-T细胞疗法等。

7. 预后

儿童胶质瘤的预后因肿瘤类型、部位和治疗方式而异：

（1）低级别胶质瘤

  • 预后较好：完全切除后可能达到长期无病生存。
  • 5年生存率：可达80%~90%。

（2）高级别胶质瘤

  • 预后较差：尤其是脑干胶质瘤（如DIPG）。
  • 5年生存率：约10%~30%。

（3）影响因素

  • 肿瘤分级：低级别胶质瘤预后优于高级别胶质瘤。
  • 手术切除程度：完全切除可显著改善预后。
  • 分子特征：BRAF融合突变患儿预后较好。

（4）后遗症

  • 神经功能障碍：运动障碍、语言障碍、认知障碍等。
  • 内分泌异常：如生长激素缺乏、甲状腺功能减退。
  • 心理影响：焦虑、抑郁等。
